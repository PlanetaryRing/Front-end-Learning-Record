使用属性选择器,选择具有某某属性的标签

例如下面的input标签,其中一个带有value属性



利用属性选择器将字体加粗变红包

/* 选择是input并且具有value属性的,用[]包含具有的属性 */
input[value]
{
 color: red;
 font-weight: bold;
}

属性选择器还可以选择指定属性值的元素

例如如下的input类型,分别为text和password,两者的type值不同,其他的均一致,利用属性选择器修改text的内容

←此时输入的文字为加粗红色
←此时输入的文字为默认
/* 选择具有某种属性,并且属性为某值的标签 */
input[type="text"]
{
 color: red;
 font-weight: bold;
}

属性选择器可以选择以某属性值开头的元素

例如,下面的四个div标签的类名依次为box1~box4,现在需要将其全部选中,并更改其颜色为红色


/* 选择以"***"属性值开头的元素 */
div[class^="box"]
{
 display: inline-block;
 width: 50px;
 height: 50px;
 margin-right: 10px;
 background-color: red;
}
/* "$="是选择以某属性值结尾的元素 */ /* "*="是选择含有字段的属性的元素 */
